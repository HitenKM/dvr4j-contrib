dvr4j
=====

针对各个主要DVR厂商SDK的Java封装

使用说明
========

leiningen:

    [dvr4j "1.0-SNAPSHOT"]

maven:

    <dependency>
      <groupId>dvr4j</groupId>
      <artifactId>dvr4j</artifactId>
      <version>1.1-SNAPSHOT</version>
    </dependency>


API文档
=======
参见[http://ruiyun.github.io/dvr4j](http://ruiyun.github.io/dvr4j)
